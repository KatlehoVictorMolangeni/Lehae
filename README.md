# 🏠 Lehae

Lehae is a student accommodation platform designed for Qwaqwa, Free State. It connects students looking for housing with landlords offering available accommodation.

---

## 🌍 Overview

Finding safe, affordable, and verified student accommodation can be difficult—especially in areas like Qwaqwa.

Lehae solves this by providing a centralized platform where:
- Students can search and compare accommodation
- Landlords can list their properties
- Listings can be filtered by price, location, and NSFAS approval
- Students can directly contact landlords

---

## 🚀 Features (MVP)

### 👨‍🎓 Students
- Browse available accommodation
- Filter by price, location, and NSFAS approval
- View detailed property listings
- Send inquiries to landlords

### 🏠 Landlords
- Create and manage listings
- Upload property images
- Set pricing and availability
- Receive student inquiries

### 🛠️ Admin
- Approve or reject listings
- Manage users
- Monitor platform activity
